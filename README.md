# Help Desk Ticket Tracker

A single-page dark-themed frontend app for creating, viewing, filtering, and updating help desk tickets.

## Features

- Create tickets with title, description, priority, and category
- View all tickets in a polished ticket queue
- Status badges for Open, In Progress, and Resolved
- Click a ticket to cycle its status
- Filter tickets by status
- Local storage support so tickets remain after refresh
- Fully contained in one HTML file

## Tech Used

- HTML
- CSS
- JavaScript
- Local Storage
